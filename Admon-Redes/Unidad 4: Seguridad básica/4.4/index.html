Defensa en Profundidad

Guía de Práctica para Redes Empresariales

1. Objetivos de la Práctica

  • Implementar la segmentación de red mediante VLANs.
  • Configurar un perímetro de seguridad con Firewall y una DMZ.
  • Aplicar Listas de Control de Acceso (ACLs) para filtrar tráfico.
  • Asegurar la gestión de dispositivos mediante protocolos seguros.

2. Topología Requerida

Arma la siguiente topología en tu espacio de trabajo de Packet Tracer:

  • Router de Borde: Conexión al ISP.
  • Firewall (ASA 5506): Núcleo de seguridad perimetral.
  • Switch de Capa 3: Para enrutar las VLANs internas.
  • DMZ: Servidor Web y Servidor DNS.
  • Red Interna: VLAN 10 (Administración), VLAN 20 (Usuarios), VLAN 30 (Servidores).

3. Fase de Configuración Técnica

Paso 1: Segmentación Lógica

enable
configure terminal
vlan 10
 name Administracion
vlan 20
 name Usuarios
vlan 30
 name Servidores_Internos

Paso 2: Seguridad Perimetral y DMZ

interface GigabitEthernet1/1
 nameif outside
 security-level 0
 ip address 200.0.0.2 255.255.255.252

interface GigabitEthernet1/2
 nameif inside
 security-level 100
 ip address 192.168.1.1 255.255.255.0

Paso 3: Listas de Control de Acceso

access-list OUTSIDE_IN extended permit tcp any host 172.16.1.10 eq 80
access-list OUTSIDE_IN extended permit tcp any host 172.16.1.10 eq 443
access-group OUTSIDE_IN in interface outside
