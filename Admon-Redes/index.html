03

Configuración Paso a Paso

Guía completa para replicar esta red en Packet Tracer

1

Abrir Cisco Packet Tracer

Inicia Cisco Packet Tracer y crea un nuevo proyecto en blanco.

File → New (Ctrl + N)
2

Agregar el Switch

Desde el panel inferior, selecciona la categoría Network Devices → Switches y arrastra un 2960 al área de trabajo.

Panel inferior → Switches → 2960-24TT
3

Agregar la Estación de Trabajo (PC)

Ve a End Devices y arrastra un PC al área de trabajo, a la izquierda del switch.

Panel inferior → End Devices → PC
4

Agregar el Servidor

Desde End Devices, arrastra un Server al área de trabajo, a la derecha del switch.

Panel inferior → End Devices → Server
5

Cablear los Dispositivos

Usa cable Copper Straight-Through para conectar:

PC0 (FastEthernet0) ──────→ Switch0 (Fa0/1)
Server0 (FastEthernet0) ──────→ Switch0 (Fa0/2)
6

Configurar IP del Servidor

Haz clic en el servidor → pestaña Desktop → IP Configuration:

IP Address192.168.1.100
Subnet Mask255.255.255.0
Default Gateway192.168.1.1
7

Activar el Servicio HTTP

En el servidor → pestaña Services → HTTP:

  1. Asegúrate de que el servicio HTTP esté en ON
  2. Edita el archivo index.html con el contenido deseado
  3. Puedes agregar más archivos HTML si lo deseas
8

Configurar IP de la Workstation

Haz clic en PC0 → pestaña Desktop → IP Configuration:

IP Address192.168.1.10
Subnet Mask255.255.255.0
Default Gateway192.168.1.1
DNS Server192.168.1.100
9

Verificar Conectividad (Ping)

En PC0 → Desktop → Command Prompt:

Command Prompt
C:\> ping 192.168.1.100

10

Probar el Servicio HTTP

En PC0 → Desktop → Web Browser:

http://192.168.1.100
